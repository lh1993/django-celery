# django-celery
django rest framework + celery 的一次容器化使用

### Requirements
- python 3.6
- Django==2.1.3
- django-celery-beat==1.3.0
- django-celery-results==1.0.4
- django-timezone-field==3.0
- djangorestframework==3.9.0
- celery==4.2.1
- celery-once==2.0.0

### Installation
```shell
pip3 install -r requirements.txt
```
